ホーム > ボタン

特徴

マウスを乗せると色が淡くなり、クリックするとボタンを押したような感覚のする、一般的なボタンUIです。
今回は例として、ダークモード切り替えのクリックイベントを付与し、一度ボタンを押すと2秒間、押せない状態になります。

ソースコードの一例

HTML

<input type="button" class="button" value="ボタン" />

CSS

input要素にはデフォルトで hover, active, disabled 時のCSSが設定されていますが、自分で設定することもできます。 

.button {
    background-color: #0f0;
    border: none;
    border-radius: 50px;
    box-shadow: 0 2px #0c0;
    padding: 8px 24px;
}

.button:hover {
    cursor: pointer;
    opacity: .6;
}

.button:active {
    box-shadow: none;
    transform: translateY(2px);
}

.button:disabled {
    box-shadow: none;
    pointer-events: none;
    transform: translateY(2px);
}

JavaScript

const button = document.getElementById("button");
button.addEventListener("click", () => {
    // この中にクリックイベントの内容を記述してください。
});

解説

  1. マウスを乗せたときに色を淡くする
    2. .button:hover {
    cursor: pointer;
    opacity: .6;
}

    マウスを乗せたときにCSSを適用するには、:hover擬似要素を使います。
    cursor: pointer; を指定するとマウスポインタが指の形になります。任意で指定してください。
    色を淡くする効果は、opacity(透明度)を上げることによって実現することができます。

  2. クリックしたときにボタンを押した感覚を持たせる
    3. .button:active {
    box-shadow: none;
    transform: translateY(2px);
}

    厳密には「ボタンを押したように見せる」というほうがニュアンス的には正しいです。
    クリックしたときにCSSを適用するには、:active擬似要素を使います。
    ① box-shadow: none; によってボタンの影をなくし、
    ② transform: translateY(2px); によってボタンの影の高さぶん(今回は2px)下に押し下げることで、視覚的にボタンを押したように見えます。

  3. ボタンが押せないときの装飾
    4. .button:disabled {
    box-shadow: none;
    pointer-events: none;
    transform: translateY(2px);
}

    pointer-events: none; を指定することで、hover, active, クリックイベントのいずれも無効になります。

  4. クリックイベントの付与
    5. button.addEventListener("click", () => {
    // この中にクリックイベントの内容を記述してください。
});

    addEventListener() の第2引数には、例のようにアロー関数を指定しても、あらかじめ定義した関数を指定することもできます。

    あらかじめ定義した関数が引数(arg)を持っている場合は、 () => doSomething(arg) のように書きましょう。
    このアロー関数は、doSomething(arg) を包んでイベント発生時に実行するための「ラッパー関数」とよびます。
