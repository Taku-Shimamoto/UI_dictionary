ホーム > ボタン

特徴

マウスを乗せると色が淡くなり、クリックするとボタンを押したような感覚のする、一般的なボタンUIです。
※押しても何も起こりませんので、好きなだけ押してください。

ソースコードの一例

HTML

<input type="button" class="button" value="ボタン" />

CSS

.button {
    background-color: #0f0;
    border: none;
    border-radius: 50px;
    box-shadow: 0 2px #0c0;
    padding: 8px 24px;
}

.button:hover {
    cursor: pointer;
    opacity: .6;
}

.button:active {
    box-shadow: none;
    transform: translateY(2px);
}

解説

  1. マウスを乗せたときに色を淡くする
    2. .button:hover {
    cursor: pointer;
    opacity: .6;
}

    マウスを乗せたときにCSSを適用するには、:hover擬似要素を使います。
    cursor: pointer; を指定するとマウスポインタが指の形になります。任意で指定してください。
    色を淡くする効果は、opacity(透明度)を上げることによって実現することができます。

  2. クリックしたときにボタンを押した感覚を持たせる
    3. .button:active {
    box-shadow: none;
    transform: translateY(2px);
}

    厳密には「ボタンを押したように見せる」というほうがニュアンス的には正しいです。
    クリックしたときにCSSを適用するには、:active擬似要素を使います。
    ① box-shadow: none; によってボタンの影をなくし、
    ② transform: translateY(2px); によってボタンの影の高さぶん(今回は2px)下に押し下げることで、視覚的にボタンを押したように見えます。
