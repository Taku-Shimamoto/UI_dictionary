特徴

Menu ボタンをクリックするとメニュー一覧が表示され、+印が回転して×印になります。
もう一度クリックするか、枠外をクリックすると元に戻ります。

ソースコードの一例

HTML

<div class="dropdown-wrapper">
    <div class="dropdown">
        <button class="dropdown-title">Menu<span></span></button>
        <ul class="menu-list">
            <li class="menu"><a href="リンク先のURL">新規登録</a></li>
            <li class="menu"><a href="リンク先のURL">ログイン</a></li>
            <li class="menu"><a href="リンク先のURL">商品一覧</a></li>
            <li class="menu"><a href="リンク先のURL">ショッピングカート</a></li>
            <li class="menu"><a href="リンク先のURL">設定</a></li>
        </ul>
    </div>
</div>

JavaScript

const dropdownTitles = Array.from(document.getElementsByClassName("dropdown-title"));
const menuLists = Array.from(document.getElementsByClassName("menu-list"));

function clearAlldropdownMenus() {
    dropdownTitles.forEach((title) => title.classList.remove("open"));
    menuLists.forEach((menu) => {
        menu.style.height = 0;
        menu.classList.remove("open");
    });
}

dropdownMenus.forEach((dropdown) => {
    dropdown.addEventListener("click", () => {
        const title = dropdown.querySelector(".dropdown-title");
        const list = dropdown.querySelector(".menu-list");
        const isOpen = list.classList.contains("open");

        clearAlldropdownMenus();

        title.classList.toggle("open", !isOpen);
        list.classList.toggle("open", !isOpen);
        list.style.height = !isOpen ? list.scrollHeight + "px" : 0;
    });
});

window.addEventListener("click", (e) => {
    const isDropdownTitle = dropdownTitles.some((title) => title.contains(e.target));
    const isMenuList = menuLists.some((menu) => menu.contains(e.target));

    if (isDropdownTitle || isMenuList) {
        return;
    }
    
    clearAlldropdownMenus();
});

解説

基本的な仕組みはアコーディオンとほとんど一緒です。合わせてご覧いただくと、より理解が深まります。

  1. 構造
    2. <div class="dropdown-wrapper">
    <div class="dropdown">
        <button class="dropdown-title">Menu<span></span></button>
        <ul class="menu-list">
            <li class="menu"><a href="リンク先のURL">新規登録</a></li>
            <li class="menu"><a href="リンク先のURL">ログイン</a></li>
            <li class="menu"><a href="リンク先のURL">商品一覧</a></li>
            <li class="menu"><a href="リンク先のURL">ショッピングカート</a></li>
            <li class="menu"><a href="リンク先のURL">設定</a></li>
        </ul>
    </div>
</div>

    開閉用の見出しボタン(<button>タグ)と、その兄弟要素である .menu-list 要素(<ul>タグ)で構成し、
    各メニューは<a>タグで作成します。

    <span>タグ内に開閉状態を表す+印をCSSで記述します。

  2. ドロップダウンメニュー見出しボタンのデザインや配置など 
    3. .dropdown-wrapper {
    margin-bottom: 35vh;
}

.dropdown {
    display: inline-block;
    position: relative;
}

.dropdown-title {
    background-color: #fff;
    border: 1px solid #000;
    border-radius: 8px;
    display: inline-flex;
    font-size: 14px;
    gap: 8px;
    justify-content: flex-start;
    margin: 16px;
    padding: 8px;
    transition: background .5s ease;
    user-select: none;
}

.dropdown-title:hover {
    background-color: #eee;
    cursor: pointer;
}

.dropdown-title span:after {
    content: "+";
    display: inline-block;
    transition: transform .5s ease;
}

.dropdown-title.open span:after {
    transform: rotate(315deg);
}

    .dropdown 要素には、ドロップダウンメニューが複数あるときに横並びになるよう、 inline-block 要素にします。
    また、メニューの表示場所がクリックしたメニューによって変わるよう、 position: relative; を指定します。

    Menu の文字と +印 を横並びにするため flex プロパティを適用します。ただし、普通の flex プロパティでは
    要素幅が画面いっぱいになってしまうので、 inline-flex を適用し flex でありながら inline-block の要素としても扱います。

    <span>タグ内に +印 を入れます。

    文字列はそのままではCSSの transform プロパティの影響を受けません。
    inline-block 要素にすることで、transform プロパティを適用できるようにしましょう。

  3. メニュー全体の配置やデザインなど
    4. .menu-list {
    background-color: #fff;
    height: 0;
    left: 16px;
    list-style: none;
    opacity: 0;
    overflow-y: hidden;
    position: absolute;
    transition: all .5s ease;
    visibility: hidden;
    width: 320px;
    z-index: 10;
}

.menu-list.open {
    opacity: 1;
    visibility: visible;
}

    アコーディオンとの共通点:
    height: 0; と overflow-y: hidden; によってデフォルト(閉じた状態)での高さとはみ出し部分をなくします。
    クリックによって .open クラスの付け外しを行い、高さの調節がされることで開閉されます。

    アコーディオンとの相違点:
    position: absolute; を指定することで、ページから「浮いた」状態になり、スクロールの計上から除外されます。
    その結果アコーディオンとは違って、ページ全体がスクロールの高さぶん押し出されなくなります。

  4. ボタンクリックによる開閉
    5. dropdownMenus.forEach((dropdown) => {
    dropdown.addEventListener("click", () => {
        const title = dropdown.querySelector(".dropdown-title");
        const list = dropdown.querySelector(".menu-list");
        const isOpen = list.classList.contains("open");

        clearAlldropdownMenus();

        title.classList.toggle("open", !isOpen);
        list.classList.toggle("open", !isOpen);
        list.style.height = !isOpen ? list.scrollHeight + "px" : 0;
    });
});

    .dropdown ブロックに対してforEach メソッドでイベントを付与し、querySelector でボタンとメニューのDOMを取得します。
    querySelector を使うことで、添字に依存せず、確実に一つのDOMを取得して動かすことができます。

    clearAlldropdownMenus() 関数によって、一度メニューの開閉状態をリセットします。こうすることで、
    他のメニューが開かれた際に前のメニューと重なって開かれることを防ぎます。

    変数 isOpen で開閉状態を判定し、閉じられていれば .open クラスを付与したうえで高さを自動計算し持たせ、
    そうでなければ .open クラスを取り除いて高さをなくします。

  5. 枠外クリックによって閉じる
    6. window.addEventListener("click", (e) => {
    const isDropdownTitle = dropdownTitles.some((title) => title.contains(e.target));
    const isMenuList = menuLists.some((menu) => menu.contains(e.target));

    if (isDropdownTitle || isMenuList) {
        return;
    }
            
    clearAlldropdownMenus();
});

    ページ全体(window)に対してクリックイベントを付与します。

    クリックした要素が「ボタン」または「メニューリスト」である場合を早期 return で除外します。
    そうでないとボタンやメニューリストをクリックした際もイベントハンドラが発火しメニューの開閉が
    正常に行われないためです。

    some メソッドはコールバック関数に対して一つでも当てはまれば true を返すものです。
    このページの例ではドロップダウンメニューは一つだけなのでなくても動きますが、複数ある場合に大きな意味を持ちます。
    クリックした要素が複数あるうちの「ボタン」または「メニューリスト」のどれかに当てはまる場合に早期 return を行う、という意味になります。

    イベント要素(event, eなどと表記されるケースが多いです)は通常、関数引数で受けるのが一般的です。
    なくてもブラウザ側が暗黙にイベント要素として処理することがありますが、現在は非推奨とされています。
    ブラウザによっては暗黙のイベント依存が壊れます。
