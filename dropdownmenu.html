ホーム > ドロップダウンメニュー
新規登録 ログイン 商品一覧 ショッピングカート 設定

特徴

Menu ボタンをクリックするとメニュー一覧が表示され、+印が回転して×印になります。
もう一度クリックすると元に戻ります。

ソースコードの一例

HTML

<button class="dropdown-title">Menu<span></span></button>
<div class="menu-wrapper">
    <a class="menu" href="リンク先のURL">新規登録</a>
    <a class="menu" href="リンク先のURL">ログイン</a>
    <a class="menu" href="リンク先のURL">商品一覧</a>
    <a class="menu" href="リンク先のURL">ショッピングカート</a>
    <a class="menu" href="リンク先のURL">設定</a>
</div>

使用する際は、aタグにhref属性を必ず指定してください。

JavaScript

const dropdownTitles = Array.from(document.getElementsByClassName("dropdown-title"));
const dropdownMenu = document.getElementById("menu_wrapper");

dropdownTitles.forEach((title) => {
    title.addEventListener("click", () => {
        title.classList.toggle("open");
        const dropdownMenu = title.nextElementSibling;
        const isOpen = dropdownMenu.classList.toggle("open");
        dropdownMenu.style.height = isOpen ? dropdownMenu.scrollHeight + "px" : 0;
    });
});

解説

基本的な仕組みはアコーディオンとほとんど一緒です。合わせてご覧いただくと、より理解が深まります。

  1. 構造
    2. <button class="dropdown-title">Menu<span></span></button>
<div class="menu-wrapper">

    開閉用の見出しボタンと、その兄弟要素である .menu-wrapper 要素で構成し、
    各メニューはaタグで作成します。

    span タグ内に開閉状態を表す+印をCSSで記述します。

  2. ドロップダウンメニュー見出しボタンのデザインや配置など 
    3. .dropdown-title {

    transform: rotate(315deg);

    Menu の文字と +印 を横並びにするため flex プロパティを適用します。ただし、普通の flex プロパティでは
    要素幅が画面いっぱいになってしまうので、 inline-flex を適用し flex でありながら inline-block の要素としても扱います。

    span タグ内に +印 を入れます。
    かつこの +印 はクリックしたときに transform が適用できるよう、inline-block 要素にします。

  3. メニュー全体の配置やデザインなど
