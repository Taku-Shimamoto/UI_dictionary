ホーム > トップスクロールボタン

少し下にスクロールしてみましょう。
右下に「Page Top」というボタンが表示されたら、押してみましょう。

特徴

少しスクロールすると表示され、クリックするとページの一番上までスクロールするボタンです。
ページの一番上にいるときは表示されないようにします。

ソースコードの一例

HTML

<div id="page_top" class="page-top">
    <a href="#">Page Top</a>
</div>

CSS

.page-top {
    opacity: 0;
    position: fixed;
    right: 10px;
    top: 90%;
    transition: all .5s;
    visibility: hidden;
    z-index: 10;
}

.page-top a {
    background-color: #00ac;
    border-radius: 8px;
    color: #fff;
    cursor: pointer;
    display: inline-block;
    font-size: 12px;
    padding: 8px 16px;
}

.page-top a:hover {
    font-weight: normal;
    opacity: .6;
    text-decoration: none;
}

JavaScript

const pageTopButton = document.getElementById("page_top");
const BUTTON_APPEARANCE_HEIGHT = 300;

// トップスクロールボタンの表示/非表示
window.addEventListener("scroll", toggleTopScrollButton);
function toggleTopScrollButton() {
    pageTopButton.style.opacity = window.scrollY > BUTTON_APPEARANCE_HEIGHT
        ? 1 : 0;
    pageTopButton.style.visibility = window.scrollY > BUTTON_APPEARANCE_HEIGHT
        ? "visible" : "hidden";
}

解説

  1. aタグのリンク先

    2. aタグのリンク先を # にすることで、ページの一番上にスクロールします。

  2. ボタンCSS
    3. .page-top {
    opacity: 0;
    position: fixed;
    right: 8px;
    top: 90%;
    transition: all .5s;
    visibility: hidden;
    z-index: 10;
}

    position: fixed; と top, right で位置を固定します。
    デフォルトはopacity: 0; + visibility: hidden; で見えないようにし、クリックもできないようにします。

    表示 / 非表示にフェード効果を与えるため、transition: all .5s; を指定します。

  3. JavaScriptでCSSを変更する
    4. // トップスクロールボタンの表示/非表示
window.addEventListener("scroll", toggleTopScrollButton);
function toggleTopScrollButton() {
    pageTopButton.style.opacity = window.scrollY > BUTTON_APPEARANCE_HEIGHT
        ? 1 : 0;
    pageTopButton.style.visibility = window.scrollY > BUTTON_APPEARANCE_HEIGHT
        ? "visible" : "hidden";
}

    windowに対してスクロールイベントを設定します。

    デフォルトではopacity: 0; , visibility: hidden; で設定されているCSSですが、スクロールの高さによって
    0 と 1 ・ hidden と visible を交互に切り替えることで、表示 / 非表示を制御することができます。
