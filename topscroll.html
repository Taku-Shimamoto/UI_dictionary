ホーム > トップスクロールボタン

少し下にスクロールしてみましょう。
右下に「Page Top」というボタンが表示されたら、押してみましょう。

特徴

少しスクロールすると表示され、クリックするとページの一番上までスクロールするボタンです。
ページの一番上にいるときは表示されないようにします。

ソースコードの一例

HTML

<div id="page_top" class="page-top">
    <a href="#">Page Top</a>
</div>

CSS

.page-top {
    opacity: 0;
    position: fixed;
    right: 10px;
    top: 90%;
    transition: all .5s;
    visibility: hidden;
    z-index: 10;
}

.page-top.is-visible {
    opacity: 1;
    visibility: visible;
}

.page-top a {
    background-color: #00ac;
    border-radius: 8px;
    color: #fff;
    cursor: pointer;
    display: inline-block;
    font-size: 12px;
    padding: 8px 16px;
}

.page-top a:hover {
    font-weight: normal;
    opacity: .6;
    text-decoration: none;
}

JavaScript

const pageTopButton = document.getElementById("page_top");
const BUTTON_APPEARANCE_HEIGHT = 300;

// トップスクロールボタンの表示/非表示
window.addEventListener("scroll", toggleTopScrollButton);
function toggleTopScrollButton() {
    pageTopButton.classList.toggle("is-visible", window.scrollY > BUTTON_APPEARANCE_HEIGHT);
}

解説

  1. aタグのリンク先

    2. aタグのリンク先を # にすることで、ページの一番上にスクロールします。

  2. ボタンCSS
    3. .page-top {
    opacity: 0;
    position: fixed;
    right: 8px;
    top: 90%;
    transition: all .5s;
    visibility: hidden;
    z-index: 10;
}

.page-top.is-visible {
    opacity: 1;
    visibility: visible;
}

    position: fixed; と top, right で位置を固定します。
    デフォルトはopacity: 0; + visibility: hidden; で見えないようにし、クリックもできないようにします。

    表示 / 非表示にフェード効果を与えるため、transition: all .5s; を指定します。

  3. 原則、JavaScriptでは「クラスの付け外し」を行うだけ
    4. // トップスクロールボタンの表示/非表示
window.addEventListener("scroll", toggleTopScrollButton);
function toggleTopScrollButton() {
    pageTopButton.classList.toggle("is-visible", window.scrollY > BUTTON_APPEARANCE_HEIGHT);
}

    デフォルトではopacity: 0; , visibility: hidden; で設定されているCSSですが、スクロールの高さによって
    .is-visible クラスが付いたり外れたりし、表示/非表示が切り替わります。

    JavaScriptで直接、CSSを変更することもできます。
    (例: pageTopButton.style.opacity = window.scrollY > BUTTON_APPEARANCE_HEIGHT ? 1 : 0;)
    ですが、CSSであらかじめ値を設定しておき、JavaScriptではクラスの付け外しを行うだけにしておくと、
    「CSSは装飾を担う」「JavaScriptはクラスの付け外しを担う」というようにそれぞれの責務の境界が明確になります。
