ホーム > スライダー(ドット)

<

>

特徴

複数の画像がスライドアニメーション形式で表示されるUIです。
一定秒数で切り替わるほか、両端のボタン操作や画像下部のドットをクリックすることで
任意の画像を表示させることもできます。

ソースコードの一例

自動切換・「前」「次」ボタン・ドットの有無などお好みに応じて使用してください。

HTML

<div class="slider-wrapper">
    <p id="prev" class="arrow prev"><</p>
    <div id="slider_track" class="slider-track">
        <img src="images/gallery01.jpg" alt="" />
        <img src="images/gallery02.jpg" alt="" />
        <img src="images/gallery03.jpg" alt="" />
        <img src="images/gallery04.jpg" alt="" />
    </div>
    <p id="next" class="arrow next">></p>
</div>
<div class="dots">
    <span class="dot active"></span>
    <span class="dot"></span>
    <span class="dot"></span>
    <span class="dot"></span>
</div>

JavaScript

const track = document.getElementById("slider_track");
const prevButton = document.getElementById("prev");
const nextButton = document.getElementById("next");
const dots = Array.from(document.getElementsByClassName("dot"));

let currentIndex = 0;
const SLIDE_WIDTH = 400;
const DURATION_MS = 4000;

function showSlide(i) {
    currentIndex = i;        
    track.style.transform = `translateX(-${currentIndex * SLIDE_WIDTH}px)`;

    dots.forEach((dot) => dot.classList.remove("active"));
    dots[currentIndex].classList.add("active");
}

setInterval(() => {
    currentIndex = (currentIndex + 1) % dots.length;
    showSlide(currentIndex);
}, DURATION_MS);

prevButton.addEventListener("click", () => {
    currentIndex = (currentIndex - 1 + dots.length) % dots.length;
    showSlide(currentIndex);
});

nextButton.addEventListener("click", () => {
    currentIndex = (currentIndex + 1) % dots.length;
    showSlide(currentIndex);
});

dots.forEach((dot, i) => dot.addEventListener("click", () => showSlide(i)));

showSlide(currentIndex);

解説

  1. 構造
    2. <div class="slider-wrapper">
    <p id="prev" class="arrow prev"><</p>
    <div id="slider_track" class="slider-track">
        <img src="images/gallery01.jpg" alt="" />
        <img src="images/gallery02.jpg" alt="" />
        <img src="images/gallery03.jpg" alt="" />
        <img src="images/gallery04.jpg" alt="" />
    </div>
    <p id="next" class="arrow next">></p>
</div>
<div class="dots">
    <span class="dot active"></span>
    <span class="dot"></span>
    <span class="dot"></span>
    <span class="dot"></span>
</div>

    スライダー要素: 画像エリア + 「前」「次」ボタン で構成します。
    ドット: ドットのみで構成します。初期状態は一枚目の画像なので、対応する1番目のドットに .active クラスを付与します。

  2. 画像エリアと「前」「次」ボタンの配置・デザイン
    3. img {
    width: 400px;
}

.slider-wrapper {
    height: 400px;
    margin: 0 auto;
    overflow: hidden;
    position: relative;
    width: 400px;
    z-index: 5;
}

.arrow {
    background-color: #fffa;
    border-radius: 50%;
    height: 24px;
    margin: 0;
    padding: 2px;
    position: absolute;
    text-align: center;
    top: 50%;
    width: 24px;
    z-index: 10; 
}

.prev {
    left: 0;
}

.next {
    right: 0;
}

.arrow:hover {
    background-color: #fffe;
    cursor: pointer;
}

.slider-track {
    display: flex;
    transition: transform .5s ease;
}

    ①画像の幅・高さと slider-wrapper の幅・高さを一致させましょう。
    ②slider-wrapper の overflow: hidden; と slider-track の flex プロパティで、はみ出し部分を隠しつつ画像を横並びにします。
    ③アニメーション秒数や ease による加減速はお好みに応じて指定してください。

  3. ドット部分
    4. .dots {
    display: flex;
    gap: 16px;
    justify-content: center;
    margin-top: 16px;
}

.dots span {
    background-color: #000;
    border-radius: 50%;
    display: inline-block;
    height: 10px;
    width: 10px;
}

.dots span:hover {
    cursor: pointer;
    opacity: .6;
}

.dots span.active {
    background-color: #f88;
}

    flex プロパティで横並び、中央寄せ、均等幅にします。
    ドットの大きさは小さすぎず、大きすぎない程度が良いでしょう。
    お好みで .active クラスが付いているドットの色を変えると「今、何番目の画像を表示しているのか」が分かりやすくなります。

  4. 共通関数
    5. function showSlide(i) {
    currentIndex = i;        
    track.style.transform = `translateX(-${currentIndex * SLIDE_WIDTH}px)`;

    dots.forEach((dot) => dot.classList.remove("active"));
    dots[currentIndex].classList.add("active");
}

showSlide(currentIndex);

    この関数が呼び起こされると、インデックス番号(何番目の画像か)が更新され、画像エリアの横幅が SLIDE_WIDTH分左にずれ、
    ドットに .active クラスを付与することで色を変えます。
    この関数を複数の方法で呼び起こすことで、画像の変更方法にバリエーションを持たせることができます。

  5. 時間経過による切り替え
    6. setInterval(() => {
    currentIndex = (currentIndex + 1) % dots.length;
    showSlide(currentIndex);
}, DURATION_MS);

    変数 currentIndex を 0→1→2→3→0 ... という形で時間経過により循環させます。

  6. 「前」「次」ボタン
    7. prevButton.addEventListener("click", () => {
    currentIndex = (currentIndex - 1 + dots.length) % dots.length;
    showSlide(currentIndex);
});

nextButton.addEventListener("click", () => {
    currentIndex = (currentIndex + 1) % dots.length;
    showSlide(currentIndex);
});

    「前」ボタン: 0→3→2→1→0 ... という形で変数 currentIndex が循環します。
    「次」ボタン: 0→1→2→3→0 ... という形で変数 currentIndex が循環します。

  7. ドットのクリックによる画像変更
    8. dots.forEach((dot, i) => dot.addEventListener("click", () => showSlide(i)));

    ドットをクリックするとドットのインデックス番号を取得し、それを showSlide() の引数に格納します。
    こうすることで任意の画像を表示することができます。
