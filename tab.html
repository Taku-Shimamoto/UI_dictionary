特徴

クリックするとそのタブの内容を表示します。
他のタブをクリックすると表示する内容を切り替えることができます。
タブ切り替えの際、背景色が横方向にスライドで移動します。

ソースコードの一例

HTML

<div class="tab-wrapper">
    <div class="tab-button-wrapper">
        <div id="tab_background" class="tab-background"></div>
        <div class="tab-button active">タブ1</div>
        <div class="tab-button">タブ2</div>
        <div class="tab-button">タブ3</div>
    </div>
    <div class="tab-display-wrapper">
        <div class="tab-display active">タブ1の内容を表示しています。</div>
        <div class="tab-display">タブ2の内容を表示しています。</div>
        <div class="tab-display">タブ3の内容を表示しています。</div>
    </div>
</div>

CSS

.tab-wrapper {
    margin-top: 40px;
    width: 480px;
}

.tab-button-wrapper {
    display: flex;
    justify-content: space-between;
    position: relative;
}

.tab-button {
    border: 2px solid #000;
    border-radius: 24px 24px 0 0;
    text-align: center;
    padding: 8px 0;
    transition: all .5s;
    width: calc(100% / 3);
    z-index: 20;

.tab-button.active {
    border-bottom: none;
    color: #fff;
}

.tab-button:hover {
    cursor: pointer;
}

.tab-button.active:hover {
    cursor: default;
}

.tab-button:not(.active):hover {
    background-color: #eee;
    transition: all .5s;
}

.tab-background {
    background-color: #00c;
    border-radius: 24px 24px 0 0;
    bottom: 0;
    left: 0;
    position: absolute;
    top: 0;
    transition: transform .5s;
    width: calc(100% / 3);
    z-index: 10;
}

.tab-display {
    border: 2px solid #000;
    border-radius: 0 0 16px 16px;
    border-top: none;
    display: none;
    height: 160px;
    padding: 8px;
}

.tab-display.active {
    display: block;
}

JavaScript

const tabButtons = Array.from(document.getElementsByClassName("tab-button"));
const tabDisplays = Array.from(document.getElementsByClassName("tab-display"));
const tabBackground = document.getElementById("tab_background");
        
let currentIndex = 0;
const TAB_WIDTH = 160;

tabButtons.forEach((tab, i) => tab.addEventListener("click", () => addClassActive(i)));
function removeClassActive() {
    tabButtons.forEach((button) => button.classList.remove("active"));
    tabDisplays.forEach((display) => display.classList.remove("active"));
}

function addClassActive(i) {
    removeClassActive();
    currentIndex = i;

    tabBackground.style.transform = `translateX(${currentIndex * TAB_WIDTH}px)`
    tabButtons[i].classList.add("active");
    tabDisplays[i].classList.add("active");
}

解説

  1. 構造
    2. <div class="tab-wrapper">
    <div class="tab-button-wrapper">
        <div id="tab_background" class="tab-background"></div>
        <div class="tab-button active">タブ1</div>
        <div class="tab-button">タブ2</div>
        <div class="tab-button">タブ3</div>
    </div>
    <div class="tab-display-wrapper">
        <div class="tab-display active">タブ1の内容を表示しています。</div>
        <div class="tab-display">タブ2の内容を表示しています。</div>
        <div class="tab-display">タブ3の内容を表示しています。</div>
    </div>
</div>

    「タブ見出し(ボタン)」 + 「タブ背景」 + 「タブ表示部分」で構成します。どちらも初期状態として、
    最初のボタンと画面を表示するために .active クラスを付与しています。

  2. タブ見出しとタブ背景の装飾やアニメーションを設定
    3. .tab-button-wrapper {
    display: flex;
    justify-content: space-between;
    position: relative;
}

.tab-button {
    border: 2px solid #000;
    border-radius: 24px 24px 0 0;
    text-align: center;
    padding: 8px 0;
    transition: all .5s;
    width: calc(100% / 3);
    z-index: 20;
}

.tab-button.active {
    border-bottom: none;
    color: #fff;
}

.tab-button:hover {
    cursor: pointer;
}

.tab-button.active:hover {
    cursor: default;
}

.tab-button:not(.active):hover {
    background-color: #eee;
    transition: all .5s;
}

.tab-background {
    background-color: #00c;
    border-radius: 24px 24px 0 0;
    bottom: 0;
    left: 0;
    position: absolute;
    top: 0;
    transition: transform .5s;
    width: calc(100% / 3);
    z-index: 10;
}

    ①子要素の div タグに対して flex プロパティを適用し、 justify-content: space-between; で均等配置にします。
    ②幅はタブの個数で等分します。今回は全体(480px)を3等分するため、 calc(100% / 3) とします。
    ③.active クラスが付いていない要素に対して反映させたくないCSSがあれば、 :not(.active) の擬似要素を使うことができます。

    CSSには「引っ張り合いの法則」というものが存在します。
    これは、背景要素 (.tab-background) の高さを、ボタン (.tab-button) の高さにぴったり合わせたい時のテクニックです。

    .tab-background に height: 40px; などと直接数値を決め打ちすると、将来タブのサイズが変わったときに
    あちこちのコードを修正することになり管理が大変になります。
    これを解決するために、position: absolute; を設定した上で、あえて top: 0; と bottom: 0; を同時に指定します。

    ブラウザは「上と下の両方にくっつきなさい」という、一見無理難題な指示を受けることになります。
    この矛盾を解決するために、ブラウザは「子要素のサイズを上下に引き伸ばす」という処理を行います。
    これが「引っ張り合いの法則」です。

    親要素「一番上にくっついてね」
    子要素「うん、わかった！」
    親要素「一番下にもくっついてちょうだい」
    子要素「ええっ！？ どっちにくっつけばいいの……？あ、そうか！僕自身がびよ〜んと伸びれば、どっちの命令も叶えられるぞ！」びよ～ん

  3. 表示画面
    4. .tab-display {
    border: 2px solid #000;
    border-radius: 0 0 16px 16px;
    border-top: none;
    display: none;
    height: 160px;
    padding: 8px;
}

.tab-display.active {
    display: block;
}

    .active クラスが付いていない要素を非表示にするため、 display: none;を指定します。
    .active クラスが付いた場合に display: block; で表示します。
    JavaScriptで直接CSSを変えることで表示/非表示にすることもできますが、JavaScriptはあくまで「動き」を制御します。

  4. JavaScriptでは「タブ背景の横移動」 + 「クラスの付け外し」を行う
    5. tabButtons.forEach((tab, i) => tab.addEventListener("click", () => addClassActive(i)));
function removeClassActive() {
    tabButtons.forEach((button) => button.classList.remove("active"));
    tabDisplays.forEach((display) => display.classList.remove("active"));
}

function addClassActive(i) {
    removeClassActive();
    currentIndex = i;

    tabBackground.style.transform = `translateX(${currentIndex * TAB_WIDTH}px)`
    tabButtons[i].classList.add("active");
    tabDisplays[i].classList.add("active");
}

    一度すべての .active クラスを取り除いてから、クリックされた要素に .active クラスを付与しましょう。

    タブ移動のしくみは画像スライダーと同じです。
    クリックされたタブのインデックス番号を取得し、変数 currentIndex に代入します。
    インデックス番号によって 0px, 160px, 320px の位置にタブ背景が移動します。
