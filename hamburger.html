ホーム > ハンバーガーメニュー

右上の三本線のアイコンをクリックしてみましょう。

特徴

三本線のアイコンです。見た目がハンバーガーの断面に似ていることからこの名前がついたと言われています。
クリックするとナビゲーションウィンドウが開き、三本線が×の形に変化します。
もう一度クリックするか、リンクをクリックすると閉じることができます。

一見難しそうに見えますが、手順はとても簡単で

  1. HTMLで構造を作成する
  2. JavaScriptで.activeクラスの付け外しをする
  3. CSSのtransitionによってアニメーションをつける

の3ステップで実装することができます。

JavaScript

const hamburgerButton = document.getElementById("hamburger");
const navigationWindow = document.getElementById("hamburger_nav");
const navigationLinks = Array.from(document.getElementsByClassName("nav-link"));

// activeクラスの付け外し
hamburgerButton.addEventListener("click", toggleClassActive);
function toggleClassActive() {
    hamburgerButton.classList.toggle("active");
    navigationWindow.classList.toggle("active");
}

// リンクをクリックしたとき、.activeクラスを消去
navigationLinks.forEach((link) => link.addEventListener("click", clearClassActive));
function clearClassActive() {
    hamburgerButton.classList.remove("active");
    navigationWindow.classList.remove("active");
}

    CSS

  3. 三本線アイコンの配置と固定
    4. .hamburger {
    cursor: pointer;
    height: 48px;
    position: fixed;
    right: 8px;
    top: 8px;
    width: 48px;
    z-index: 30;
}

.hamburger span:nth-of-type(1) {
    top: 16px;
}

.hamburger span:nth-of-type(2) {
    top: 24px;
}

.hamburger span:nth-of-type(3) {
    top: 32px;
}

    position: fixed; で固定し、z-index を指定して他の要素よりも上にくるように配置します。
    ここでは、ヘッダーを10、ナビゲーションウィンドウを20、三本線アイコンを30とし、
    ヘッダー < ナビゲーションウィンドウ < 三本線アイコン の順で表示されるようにします。

    cursor: pointer; は好みに応じて指定してください。

    :nth-of-type(1 ~ 3) を指定することで、1本目・2本目・3本目を別々の高さに指定します。

  4. 三本線アイコンのデザイン
    5. .hamburger span {
    background-color: #fff;
    border-radius: 50px;
    height: 2px;
    position: absolute;
    transition: all .5s;
    width: 32px;
}

    ここでポイントになるのが、transition: all .5s; の部分です。
    三本線が×の形に変化するための秒数をここで指定することで、クリックしたときに滑らかに動くようになります。

  5. ナビゲーションウィンドウの配置とデザイン
    6. header nav {
    background-color: #333c;
    display: flex;
    height: 100vh;
    justify-content: center;
    position: fixed;
    right: -30%;
    top: 0;
    transition: all .5s;
    width: 30%;
    z-index: 20;
}

header nav ul {
    list-style: none;
    line-height: 200%;
    margin-top: 48px;
    overflow: auto;
    width: 90%;
}

header nav a {
    color: #fff;
}

    ナビゲーションウィンドウの背景色に透明度を入れることで、コンテンツが完全に隠れないようにします。
    高さを100vhに指定することで、ナビゲーションウィンドウの高さが画面いっぱいになります。
    rightの値をwidthの値が対応するようにします。これは、幅30%ぶんの要素が、画面右端のさらに30%右側に配置されるという意味です。
    つまり、この時点では画面右端の見えない部分に「隠している」ということになります。

  6. 三本線アイコンをクリックしたときの挙動
    7. header nav.active {
    right: 0;
}

.hamburger.active span:nth-of-type(1) {
    top: 24px;
    transform: rotate(-45deg);
}

.hamburger.active span:nth-of-type(2) {
    opacity: 0;
}

.hamburger.active span:nth-of-type(3) {
    top: 24px;
    transform: rotate(45deg);
}

    三本線アイコンをクリックすると、JavaScriptによって.activeクラスが付与されます(後述)。
    「5. ナビゲーションウィンドウの配置とデザイン」で述べた通り、ナビゲーションウィンドウは-30%の幅の分だけ、画面右端に隠れている状態です。
    .activeクラスが付いている時のみ、これを right: 0; にすることで、隠れていたナビゲーションウィンドウが現れるようになります。

    1本目・3本目の線にはそれぞれ-45°・45°の傾斜をつけることで、×の形を作ります。
    2本目の線にはopacity: 0; を指定することで、一時的に見えない状態にします。

    注意点として、 header nav.active のように、navと.activeとの間にスペースを入れないようにしましょう。
    スペースを入れてしまうと、「ヘッダー内のnavタグ内にある.activeクラスが付いている要素」という意味になり、
    今回のように「要素に.activeクラスが付与されたとき」のニュアンスとはズレてしまいます。

    JavaScript

  7. JavaScriptで行うのは「.activeクラスの付け外し」のみ
    8. // activeクラスの付け外し
hamburgerButton.addEventListener("click", toggleClassActive);
function toggleClassActive() {
    hamburgerButton.classList.toggle("active");
    navigationWindow.classList.toggle("active");
}

// リンクをクリックしたとき、.activeクラスを消去
navigationLinks.forEach((link) => link.addEventListener("click", clearClassActive));
function clearClassActive() {
    hamburgerButton.classList.remove("active");
    navigationWindow.classList.remove("active");
}

    三本線アイコンをクリックしたときはtoggleメソッドを使い、「.activeクラスがなければ付与し、あれば取り除く」ようにします。
    一方でリンクをクリックしたときに自動でナビゲーションウィンドウが閉じられるよう、それぞれのリンクにはremoveメソッドを
    用いて必ず.activeクラスを取り除くようにします。
