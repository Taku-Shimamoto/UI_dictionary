ホーム > ハンバーガーメニュー

右上の三本線のアイコンをクリックしてみましょう。

特徴

三本線のアイコンです。見た目がハンバーガーの断面に似ていることからこの名前がついたと言われています。
クリックするとナビゲーションウィンドウが開き、三本線が×の形に変化します。
もう一度クリックするか、リンクをクリックすると閉じることができます。

一見難しそうに見えますが、手順はとても簡単で

  1. HTMLで構造を作成する
  2. JavaScriptで.activeクラスの付け外しをする
  3. CSSのtransitionによってアニメーションをつける

の3ステップで実装することができます。

JavaScript

const hamburgerButton = document.getElementById("hamburger");
const navigationWindow = document.getElementById("hamburger_nav");
const navigationLinks = Array.from(document.getElementsByClassName("nav-link"));

hamburgerButton.addEventListener("click", toggleClassActive);
function toggleClassActive() {
    hamburgerButton.classList.toggle("active");
    navigationWindow.classList.toggle("active");
}

navigationLinks.forEach((link) => link.addEventListener("click", clearClassActive));
function clearClassActive() {
    hamburgerButton.classList.remove("active");
    navigationWindow.classList.remove("active");
}

window.addEventListener("click", (e) => {
    const isHumburgerButton = hamburgerButton.contains(e.target);
    const isHumburgerWindow = navigationWindow.contains(e.target);

    if (isHumburgerButton || isHumburgerWindow) return;
    clearClassActive();
});

  3. 三本線アイコンの配置と固定
    4. .hamburger {
    cursor: pointer;
    height: 48px;
    position: fixed;
    right: 8px;
    top: 8px;
    width: 48px;
    z-index: 30;
}

.hamburger span:nth-of-type(1) {
    top: 16px;
}

.hamburger span:nth-of-type(2) {
    top: 24px;
}

.hamburger span:nth-of-type(3) {
    top: 32px;
}

    position: fixed; で固定し、z-index を指定して他の要素よりも上にくるように配置します。
    ここでは、ヘッダーを10、ナビゲーションウィンドウを20、三本線アイコンを30とし、
    ヘッダー < ナビゲーションウィンドウ < 三本線アイコン の順で表示されるようにします。

    cursor: pointer; は好みに応じて指定してください。

    :nth-of-type(1 ~ 3) を指定することで、1本目・2本目・3本目を別々の高さに指定します。

  4. 三本線アイコンのデザイン
    5. .hamburger span {
    background-color: #fff;
    border-radius: 50px;
    height: 2px;
    position: absolute;
    transition: all .5s;
    width: 32px;
}

    ここでポイントになるのが、transition: all .5s; の部分です。
    三本線が×の形に変化するための秒数をここで指定することで、クリックしたときに滑らかに動くようになります。

  5. ナビゲーションウィンドウの配置とデザイン
    6. header nav {
    background-color: #333c;
    display: flex;
    height: 100vh;
    justify-content: center;
    position: fixed;
    right: 0;
    top: 0;
    transform: translateX(100%);
    transition: all .5s;
    width: calc(100% / 5);
    z-index: 20;
}

header nav ul {
    list-style: none;
    line-height: 200%;
    margin-top: 48px;
    overflow: auto;
    width: 90%;
}

header nav a {
    color: #fff;
}

    ナビゲーションウィンドウの背景色は好みで指定して構いませんが、コンテンツの一部が隠れることによるUX低下に配慮し、
    ある程度の透明度を入れることも好まれます。

    height の値を 100vh に指定することで、ページの高さいっぱいに合わせます。

    非表示の状態は厳密に言うと「画面の右側に隠している」という状態です。これは right: 0; と transform: translateX(100%);
    の指定によってナビゲーションウィンドウの幅のぶんだけ右端からさらに右にズラしており、これが結果として画面の右側に隠れているということになります。

    ナビゲーションウィンドウの幅は表示する内容によって調整が必要です。
    今回は calc(100% / 5) によって画面全体の5分の1の幅という指定にしています。

  6. 三本線アイコンをクリックしたときの挙動
    7. header nav.active {
    transform: translateX(0);
}

.hamburger.active span:nth-of-type(1) {
    top: 24px;
    transform: rotate(-45deg);
}

.hamburger.active span:nth-of-type(2) {
    opacity: 0;
}

.hamburger.active span:nth-of-type(3) {
    top: 24px;
    transform: rotate(45deg);
}

    JavaScript によって .active クラスが付与されたとき、画面右側に隠していたナビゲーションウィンドウを画面内に移動します。

    1本目・3本目の線にはそれぞれ-45°・45°の傾斜をつけることで、×の形を作ります。
    2本目の線には opacity: 0; を指定することで、一時的に見えない状態にします。

    注意点として、 header nav.active のように、nav と .active との間にスペースを入れないようにしましょう。
    スペースを入れてしまうと、「ヘッダーの<nav>タグ内にある .active クラスが付いている要素」という意味になり、
    今回のように「要素に .active クラスが付与されたとき」のニュアンスとはズレてしまいます。

  7. JavaScript で行うのは「.active クラスの付け外し」のみ
    8. hamburgerButton.addEventListener("click", toggleClassActive);
function toggleClassActive() {
    hamburgerButton.classList.toggle("active");
    navigationWindow.classList.toggle("active");
}

navigationLinks.forEach((link) => link.addEventListener("click", clearClassActive));
function clearClassActive() {
    hamburgerButton.classList.remove("active");
    navigationWindow.classList.remove("active");
}

    三本線アイコンをクリックしたときは toggle メソッドを使い、.active クラスがなければ付与し、あれば取り除くようにします。
    一方でリンクをクリックしたときに自動でナビゲーションウィンドウが閉じられるよう、それぞれのリンクには remove メソッドを
    用いて必ず .active クラスを取り除くようにします。

  8. 外部クリックでも閉じるように設定(UX)
    9. window.addEventListener("click", (e) => {
    const isHumburgerButton = hamburgerButton.contains(e.target);
    const isHumburgerWindow = navigationWindow.contains(e.target);

    if (isHumburgerButton || isHumburgerWindow) return;
    clearClassActive();
});

    ドロップダウンメニューと同様、window に対してクリックイベントを付与して .active クラスを取り除きます。
    この際、クリックした要素が「ハンバーガーアイコン」と「ナビゲーションウィンドウ」である場合に早期 return をしてクリックイベントが
    発火しないようにしましょう。
