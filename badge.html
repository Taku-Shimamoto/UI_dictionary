ホーム > 購入バッジ

好きな商品を選び、「カートに入れる」ボタンを押してみましょう。

個

特徴

ECサイトなどでよく見かける購入バッジです。
カートに商品が1つ以上あるとき、カートアイコンの右上に個数を表す赤いバッジアイコンが表示されます。
ここでは、「9」を表示上限とし、それ以上の商品がカートに入っている場合は「9+」と表示します。

購入バッジの中に表示する購入数の取得方法については、使用する言語に依存しますので、ここでは言及しません。

ソースコードの一例

HTML

カートのアイコンはFont Awesomeより拝借しています。気になる方は調べてみましょう。

CSS

.cart-icon-wrapper {
    position: relative;
}

.qty-badge {
    background-color: #f00;
    border-radius: 50%;
    color: #fff;
    cursor: default;
    display: grid;
    font-size: 12px;
    height: 24px;
    place-items: center;
    position: absolute;
    right: 0;
    top: 0;
    transform: translate(50%, -50%);
    user-select: none;
    width: 24px;
    z-index: 5;
}

.cart-icon {
    color: #fff;
    font-size: 24px;
    line-height: 48px;
}

.cart-icon:hover {
    cursor: pointer;
}

解説

  1. 購入バッジのデザイン、配置など
    2. .qty-badge {
    background-color: #f00;
    border-radius: 50%;
    color: #fff;
    cursor: default;
    display: grid;
    font-size: 12px;
    height: 24px;
    place-items: center;
    position: absolute;
    right: 0;
    top: 0;
    transform: translate(50%, -50%);
    user-select: none;
    width: 24px;
    z-index: 5;
}

    ツールチップと同様、grid レイアウト + place-items: center; を指定してバッジの中の数字を縦横ともに中央配置にします。

    特に重要なのが transform: translate(50%, -50%); の部分です。
    これは「要素の50%分右と上にズラす」という意味で、これを指定することでアイコンの高さ・幅・大きさなどに
    依存せず常に右上の位置に配置することができます。

  2. 在庫「0」のときは表示しない

    3. 扱う言語にもよりますが、if文や三項演算子などを使って、カート内の数量を表す変数が0のときとそうでないときとで
    表示を変えるようにしましょう。

    PHPなどのサーバーサイド言語では、数量に加えてログイン状態(セッション状態)なども参照しておくと、
    「ログアウトしているのに購入バッジが表示されている」などの事故を防ぐことができます。
