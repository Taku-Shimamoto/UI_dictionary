ホーム > モーダルウィンドウ
モーダルを表示します。

特徴

画面中央にウィンドウと×印を表示します。
表示されている間は背景が暗転し、その間はスクロールができなくなります。
モーダルウィンドウの×印を押すか、Escキーを押下するとモーダルウィンドウを閉じることができます。

JavaScript

const modalButton = document.getElementById("modal_button");
const modalBackground = document.getElementById("modal_background");

function getScrollbarWidth() {
    return window.innerWidth - document.documentElement.clientWidth;
}

modalButton.addEventListener("click", openModal);
function openModal() {
    const scrollbarWidth = getScrollbarWidth();
    document.body.style.paddingRight = `${scrollbarWidth}px`;
    modalBackground.classList.add("open");
    document.body.classList.add("no-scroll");
}

const closeButton = document.getElementById("close");
closeButton.addEventListener("click", closeModal);
document.addEventListener("keydown", (e) => {
    if(e.key === "Escape") {
        closeModal();
    }
});
function closeModal() {
    document.body.style.paddingRight = "";
    modalBackground.classList.remove("open");
    document.body.classList.remove("no-scroll");
}

解説

  1. モーダルの仕組み
    2. <input type="button" id="modal_button" class="modal-button" value="クリックしてみましょう" />

    モーダルの仕組みは「ボタン」「背景」「モーダルウィンドウ」の3つになります。
    ボタンと背景は兄弟要素、背景とモーダルウィンドウは親子要素にします。

  2. ボタンCSS

    transform: translateY(2px);
}

    ボタンの装飾です。好きなデザインのものがあればそれを使っていただいて構いません。

  3. モーダルの背景
    4. .modal-background {
    align-items: center;
    background-color: #000c;
    display: flex;
    height: 100vh;
    justify-content: center;
    left: 0;
    opacity: 0;
    position: fixed;
    top: 0;
    transition: all .8s;
    visibility: hidden;
    width: 100%;
    z-index: 10;
}

.modal-background.open {
    opacity: 1;
    visibility: visible;
}

    ①モーダルウィンドウを画面中央に配置するためのflexプロパティを、ここで指定します。
    　(display: flex; , align-items: center; , justify-content: center;)

    ②背景色に透明度を持たせ、高さと幅をそれぞれ画面いっぱいにします。
    　position: fixed; で位置も固定します。

    ③opacity: 0; と visibility: hidden; で見えないようにし、 transition: all .8s; でアニメーション秒数を指定します。
    　JavaScriptで .open クラスの付け外しをした際に opacity と visibility の値が変化して表示/非表示を制御することができます。

    モーダルウィンドウの×印のデザインは、ハンバーガーメニューのものと一緒です。

  5. モーダルウィンドウ表示中のスクロール無効
    6. .no-scroll {
    overflow: hidden;
}

    JavaScriptでbodyタグに .no-scroll クラスが付与されると、スクロールを無効にします。

  6. 原則、JavaScriptでは「クラスの付け外し」を行うだけ
    7. function getScrollbarWidth() {
    return window.innerWidth - document.documentElement.clientWidth;
}

modalButton.addEventListener("click", openModal);
function openModal() {
    const scrollbarWidth = getScrollbarWidth();
    document.body.style.paddingRight = `${scrollbarWidth}px`;
    modalBackground.classList.add("open");
    document.body.classList.add("no-scroll");
}

const closeButton = document.getElementById("close");
closeButton.addEventListener("click", closeModal);
document.addEventListener("keydown", (e) => {
    if(e.key === "Escape") {
        closeModal();
    }
});
function closeModal() {
    document.body.style.paddingRight = "";
    modalBackground.classList.remove("open");
    document.body.classList.remove("no-scroll");
}

    モーダルウィンドウに .open クラスを付与して表示、 body タグに .no-scroll クラスを付与してスクロールを無効にします。
    閉じるときも同様に .open クラスと .no-scroll クラスを取り除きます。

    原則、JavaScriptで直接CSSの値を変更しないほうが、「CSS→装飾、JavaScript→クラス付け外し」という責務の境界を守る
    という意味で望ましいですが、今回は「スクロールバーの幅ぶんの padding を操作する」ことをしています。
    こうすることで、モーダルウィンドウ表示時にスクロールバーの幅ぶんズレるという動きを防ぎます。
