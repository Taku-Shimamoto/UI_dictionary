「？」のアイコンにマウスを乗せてみましょう。

ONにすると、すべてのユーザーがあなたの投稿を閲覧することができます。
OFFにすると、あなたが承認したユーザーのみがあなたの投稿を閲覧することができます。
ONにすると、すべてのユーザーがあなたに対してメッセージを送ることができます。

※このトグルスイッチはダミーです。押しても何も起こりません。

特徴

「？」マークのアイコンにマウスを乗せると、その間だけ説明文が表示されます。
マウスが離れると、表示されていた説明文は非表示になります。

ソースコードの一例

HTML

<div class="setting">
    <label>アカウントを公開する<span class="tooltip-icon">？</span></label>
    <div class="tooltip-info">
        ONにすると、すべてのユーザーがあなたの投稿を閲覧することができます。<br />
        OFFにすると、あなたが承認したユーザーのみがあなたの投稿を閲覧することができます。
    </div>
</div>

CSS

.setting {
    margin-top: 16px;
    position: relative;
}

.tooltip-icon {
    background-color: #000;
    border-radius: 50%;
    color: #fff;
    display: inline-block;
    font-size: 12px;
    font-weight: bold;
    height: 24px;
    margin-left: 32px;
    padding: 2px;
    text-align: center;
    user-select: none;
    vertical-align: text-top;
    width: 24px;
}

.tooltip-icon:hover {
    cursor: pointer;
}

.tooltip-info {
    background-color: #333a;
    border-radius: 8px;
    color: #fff;
    display: inline-block;
    font-size: 12px;
    opacity: 0;
    padding: 8px;
    position: absolute;
    transform: translate(5%, -50%);
    transition: 
        opacity .2s ease,
        visibility .2s ease;
    visibility: hidden;
}

.tooltip-info.active {
    opacity: 1;
    visibility: visible;
}

JavaScript

const settings = document.querySelectorAll(".setting");

settings.forEach((setting) => {
    const icon = setting.querySelector(".tooltip-icon");
    const info = setting.querySelector(".tooltip-info");
        
    icon.addEventListener("mouseenter", () => {
        info.classList.add("active");
    });
        
    icon.addEventListener("mouseleave", () => {
        info.classList.remove("active");
    });
});

解説

  1. アイコンのデザイン
    2. .tooltip-icon {
    background-color: #000;
    border-radius: 50%;
    color: #fff;
    display: inline-block;
    font-size: 12px;
    font-weight: bold;
    height: 24px;
    margin-left: 32px;
    padding: 2px;
    text-align: center;
    user-select: none;
    vertical-align: text-top;
    width: 24px;
}

.tooltip-icon:hover {
    cursor: pointer;
}

    vertical-align の値がデフォルトだと、文字の高さに対してやや下目に配置されます。
    今回は text-top を指定して親要素の上端とアイコンの上端が揃うように配置しています。

    あるいは position: relative; と position: absolute; を指定して、任意の位置に配置することもできます。
    お好みで調整をしてください。

  2. テキストのデザイン、配置、アニメーションの指定
    3. .tooltip-info {
    background-color: #333a;
    border-radius: 8px;
    color: #fff;
    display: inline-block;
    font-size: 12px;
    opacity: 0;
    padding: 8px;
    position: absolute;
    transform: translate(5%, -50%);
    transition: 
        opacity .2s ease,
        visibility .2s ease;
    visibility: hidden;
}

.tooltip-info.active {
    opacity: 1;
    visibility: visible;
}

    背景色はお好みで指定して構いませんが、背景色が他の要素を見えなくすることによるUX低下に配慮して、
    ある程度の透明度を加えたり、ページデザインに合わせた色合いを選んだりする傾向が好まれます。

    transform: traslate(5%, -50%); は右にアイコンの5%ぶん、上にアイコンの50%ぶんズラすという意味です。
    表示される説明文がアイコンより少し右、アイコンの上端に説明文の縦中央が合うように配置されています。

  3. JavaScriptで .active クラスの付け外しを行う
    4. settings.forEach((setting) => {
    const icon = setting.querySelector(".tooltip-icon");
    const info = setting.querySelector(".tooltip-info");
        
    icon.addEventListener("mouseenter", () => {
        info.classList.add("active");
    });
        
    icon.addEventListener("mouseleave", () => {
        info.classList.remove("active");
    });
});

    .setting のDOMを取得し forEach メソッドを使用する理由は、対応するツールチップのアイコンと説明文のDOMを
    添字(インデックス番号)に依存せず、querySelector を使用し確実に一つだけ取得したいためです。

    mouseenter イベントに似たイベントリスナーとして mouseover イベントがありますが、これは親要素の hover で
    イベントが発火したあと、子要素に hover するたびにイベントが発火します。将来的に子要素が増えてくると、これが
    意図しない挙動になる可能性も出てきます。そういう場合は mouseenter イベントが好まれます。

    最近はスマートフォンやタブレット端末が普及しているため、pointerenter や pointerleave イベントなども好まれます。
