ホーム > トースト通知

Copy ボタンを押して、ソースコードをコピーしてみましょう。

特徴

何か処理をおこなったとき、その処理の結果を伝えるちょっとした通知です。
ここでは、画面右側からそっと通知が現れ、2秒後にそっと画面外に戻っていきます。

画面の端から出てくる様子がトースターから飛び出す食パンのようであることから、この名前が付いたそうです。

ソースコードの一例

HTML

<div id="toast_notice" class="toast-notice">
    ✓ ここに通知として表示するメッセージを記述してください。
</div>

CSS

.toast-notice {
    background-color: #0b0b;
    border-radius: 8px;
    color: #fff;
    cursor: default;
    display: inline-block;
    font-size: 14px;
    padding: 8px 16px;
    position: fixed;
    right: 0;
    top: 72px;
    transform: translateX(100%);
    transition: transform .5s ease;
    user-select: none;
}

.toast-notice.active {
    transform: translateX(-8px);
}

JavaScript

const toastNotice = document.getElementById("toast_notice");
const RETURN_MS = 3000;
let toastTimerId = null;

function toggleToastActive() {
    if (toastTimerId) {
        clearTimeout(toastTimerId);
    }

    toastNotice.classList.add("active");
    toastTimerId = setTimeout(() => {
        toastNotice.classList.remove("active");
        toastTimerId = null;
    }, RETURN_MS);
}

// コードをペーストしただけでは動きません。
// toggleToastActive() 関数を、クリックイベントと紐づける等して呼び出しましょう。

解説

  1. トースト通知のデザインと配置
    2. .toast-notice {
    background-color: #0b0b;
    border-radius: 8px;
    color: #fff;
    cursor: default;
    display: inline-block;
    font-size: 14px;
    padding: 8px 16px;
    position: fixed;
    right: 0;
    top: 72px;
    transform: translateX(100%);
    transition: transform .5s ease;
    user-select: none;
}

.toast-notice.active {
    transform: translateX(-8px);
}

    top の値はこのページのヘッダーの高さ(64px)に合わせて72pxになっています。
    72px = 64px(ヘッダーの高さ) + 8px(余白)
    実際のページレイアウトに合わせて、お好みで高さを調整してください。

    通常時は通知が見えないよう、 right: 0; と transform: translateX(100%); で通知の幅100%ぶんを画面右側に隠しておきます。
    JavaScript で .active クラスを付与したときに、画面内にスライドするようにします。
    画面右端との余白が必要なければ、translateX() の値は0でもかまいません。

  2. JavaScriptでの .active クラスの付け外し
    3. function toggleToastActive() {
    if (toastTimerId) {
        clearTimeout(toastTimerId);
    }

    toastNotice.classList.add("active");
    toastTimerId = setTimeout(() => {
        toastNotice.classList.remove("active");
        toastTimerId = null;
    }, RETURN_MS);
}

    ほかのUIと同様、JavaScript では .active クラスの付け外しを行います。

    タイマーIDを使用して、処理を行う前にタイマーリセットをしておくと、例えば連続でクリックした場合、
    最後に通知を表示してから2秒後に通知が引っ込むようになります。
    タイマーリセットをしないと、連続クリックをしても最初の通知が出てから2秒後に強制的に引っ込んでしまいます。
