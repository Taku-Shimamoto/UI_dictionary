ホーム > アコーディオン

ここをクリックしてみましょう。

クリックすると中身が表示されます。

特徴

見出し部分をクリックすると中身が表示され、+マークから-マークに変化します。
もう一度クリックすると中身が格納され、-マークから+マークに戻ります。

ソースコードの一例

HTML

<p class="accordion-title">ここをクリックしてみましょう。<span></span></p>
<div class="accordion">
    <p>クリックすると中身が表示されます。</p>
</div>

CSS

.accordion-title {
    display: flex;
    justify-content: space-between;
    width: 40%;
}

.accordion-title:hover {
    cursor: pointer;
}

.accordion-title span:after {
    content: "+";
}

.accordion-title.open span:after {
    content: "-";
}

.accordion {
    height: 0;
    overflow-y: hidden;
    transition: all .5s;
}

JavaScript

const accordionTitles = Array.from(document.getElementsByClassName("accordion-title"));
const accordions = Array.from(document.getElementsByClassName("accordion"));

accordionTitles.forEach((title, i) => title.addEventListener("click", () => toggleClassOpen(i)));
function toggleClassOpen(i) {
    accordionTitles[i].classList.toggle("open");
    const isOpen = accordions[i].classList.toggle("open");
    accordions[i].style.height = isOpen ? accordions[i].scrollHeight + "px" : "0";
}

解説

  1. アコーディオンの高さ・幅・アニメーション秒数の設定
    2. .accordion {
    height: 0;
    overflow-y: hidden;
    transition: all .5s;
}

    クリックしていない状態で中身が表示されないよう、高さを0、縦方向のはみ出し部分を hidden に設定します。
    アニメーション秒数は好みに応じて調整しましょう。

  2. .open クラスの付け外しと高さの取得
    3. accordionTitles.forEach((title, i) => title.addEventListener("click", () => toggleClassOpen(i)));
function toggleClassOpen(i) {
    accordionTitles[i].classList.toggle("open");
    const isOpen = accordions[i].classList.toggle("open");
    accordions[i].style.height = isOpen ? accordions[i].scrollHeight + "px" : "0";
}

    アコーディオンの見出しと中身の両方に .open クラスの付け外しを行います。
    isOpen はアコーディオンの中身に .open クラスの付け外しを行い、クラス名が付いていればtrueを、そうでなければfalseを返します。
    .open クラス名が付いていればその要素の高さを取得し指定することで中身が表示されます。
    そうでなければ0が指定され、中身が隠れます。
