ホーム > アコーディオン
ここをクリックしてみましょう。

クリックすると中身が表示されます。

特徴

見出し部分をクリックすると中身が表示され、+マークが回転して×マークに変化します。
もう一度クリックすると中身が格納され、×マークから+マークに戻ります。

ソースコードの一例

HTML

<div class="accordion-wrapper">
    <div class="accordion-title">ここをクリックしてみましょう。<span></span></div>
    <div class="accordion-text">
        <p>
            クリックすると中身が表示されます。
        </p>
    </div>
</div>

CSS

.accordion-wrapper {
    margin: 16px;
    width: 400px;
}

.accordion-title {
    display: flex;
    font-size: 18px;
    justify-content: space-between;
}

.accordion-title:hover {
    cursor: pointer;
}

.accordion-title span:after {
    content: "+";
    display: inline-block;
    transition: transform .5s;
}

.accordion-title.open span:after {
    transform: rotate(315deg);
}

.accordion-text {
    height: 0;
    overflow-y: hidden;
    transition: all .5s;
}

JavaScript

const accordions = Array.from(document.querySelectorAll(".accordion-wrapper"));

accordions.forEach((accordion) => {
    accordion.addEventListener("click", () => {
        const title = accordion.querySelector(".accordion-title");
        const text = accordion.querySelector(".accordion-text");

        const isTitleOpen = title.classList.toggle("open");
        const isTextOpen = text.classList.toggle("open");
        text.style.height = (isTitleOpen && isTextOpen) ? text.scrollHeight + "px" : 0;
    });
});

解説

  1. アコーディオンの+マークを×マークへ変化
    2. .accordion-title span:after {
    content: "+";
    display: inline-block;
    transition: transform .5s;
}

.accordion-title.open span:after {
    transform: rotate(315deg);
}

    transform: rotate(315deg); を指定して回転させるのですが、+マークは文字列であるため、そのままでは transition の対象になりません。
    transition の対象にするために inline-block 要素に指定します。

  2. アコーディオンの高さ・幅・アニメーション秒数の設定
    3. .accordion {
    height: 0;
    overflow-y: hidden;
    transition: all .5s;
}

    クリックしていない状態で中身が表示されないよう、高さを0、縦方向のはみ出し部分を hidden に設定します。
    アニメーション秒数は好みに応じて調整しましょう。

  3. .open クラスの付け外しと高さの取得
    4. accordions.forEach((accordion) => {
    accordion.addEventListener("click", () => {
        const title = accordion.querySelector(".accordion-title");
        const text = accordion.querySelector(".accordion-text");

        const isTitleOpen = title.classList.toggle("open");
        const isTextOpen = text.classList.toggle("open");
        text.style.height = (isTitleOpen && isTextOpen) ? text.scrollHeight + "px" : 0;
    });
});

    アコーディオン全体を包むラッパー要素(.accordion-wrapper)に対してDOMを取得し、forEach メソッドでイベントを付与します。
    querySelector を使って添字に依存しない + 確実に一つの見出しとテキストDOMを取得する、という目的があります。

    toggle メソッドを使って .open クラスの付け外しを行い、状態としてとっておきます。
    どちらも .open クラスがある場合(開いている場合)、scrollHeight によって自動で高さを取得します。
    .open クラスがない場合(アコーディオンを閉じている場合)、高さは0になり、中身を隠します。
