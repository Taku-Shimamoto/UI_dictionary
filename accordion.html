ホーム > アコーディオン
ここをクリックしてみましょう。

クリックすると中身が表示されます。

特徴

見出し部分をクリックすると中身が表示され、+マークから-マークに変化します。
もう一度クリックすると中身が格納され、-マークから+マークに戻ります。

ソースコードの一例

HTML

<div class="accordion-title">ここをクリックしてみましょう。<span></span></div>
<div class="accordion">
    <p>クリックすると中身が表示されます。</p>
</div>

CSS

.accordion-title {
    display: flex;
    justify-content: space-between;
    width: 40%;
}

.accordion-title:hover {
    cursor: pointer;
}

.accordion-title span:after {
    content: "+";
}

.accordion-title.open span:after {
    content: "-";
}

.accordion {
    height: 0;
    overflow-y: hidden;
    transition: all .5s;
}

JavaScript

const accordionTitles = Array.from(document.getElementsByClassName("accordion-title"));

accordionTitles.forEach((title) => {
    title.addEventListener("click", () => {
        title.classList.toggle("open");

        const accordion = title.nextElementSibling;
        const isOpen = accordion.classList.toggle("open");

        accordion.style.height = isOpen ? accordion.scrollHeight + "px" : 0;
    });
});

解説

  1. アコーディオンの高さ・幅・アニメーション秒数の設定
    2. .accordion {
    height: 0;
    overflow-y: hidden;
    transition: all .5s;
}

    クリックしていない状態で中身が表示されないよう、高さを0、縦方向のはみ出し部分を hidden に設定します。
    アニメーション秒数は好みに応じて調整しましょう。

  2. .open クラスの付け外しと高さの取得
    3. accordionTitles.forEach((title) => {
    title.addEventListener("click", () => {
        title.classList.toggle("open");

        const accordion = title.nextElementSibling;
        const isOpen = accordion.classList.toggle("open");

        accordion.style.height = isOpen ? accordion.scrollHeight + "px" : 0;
    });
});

    アコーディオンの見出しに .open クラスの付け外しを行います。

    nextElementSibling によって見出しの兄弟要素である .accordion 要素を取得します。

    isOpen はアコーディオンの中身に .open クラスの付け外しを行い、クラス名が付いていればtrueを、そうでなければfalseを返します。
    .open クラス名が付いていればその要素の高さを取得し指定することで中身が表示されます。
    そうでなければ0が指定され、中身が隠れます。
