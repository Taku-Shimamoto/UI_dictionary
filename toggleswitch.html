ホーム > トグルスイッチ

クリックしてみましょう。

ダークモードにする ON / OFF

太字にする ON / OFF

特徴

クリックすると内側の丸いボタンが右側に動き、スイッチ背面の色が灰色→緑色に変化します。
もう一度クリックすると元の状態に戻ります。

ソースコードの一例

このページの「ダークモード切り替え(darkmode)」「太字切り替え(boldtext)」はあくまで一例です。
data-setting 属性を自分好みに変更して、ぜひ自分の好きな機能を実装してみてください！

HTML

<div class="switch-outer" data-setting="darkmode">
    <div class="toggle-switch"></div>
</div>

<div class="switch-outer" data-setting="boldtext">
    <div class="toggle-switch"></div>
</div>

CSS

body.darkmode {
    background-color: #333;
    color: #fff;
}

body.boldtext {
    font-weight: bold;
}

.switch-outer {
    background-color: #ddd;
    border-radius: calc(infinity * 1px);
    height: 40px;
    margin: 8px 0;
    position: relative;
    transition: all .5s ease;
    width: 80px;
}

.switch-outer:hover {
    cursor: pointer;
}

.switch-outer.active {
    background-color: #0f0;
}

.toggle-switch {
    background-color: #fff;
    border-radius: 50%;
    bottom: 0;
    box-shadow: 1px 1px 8px #ccc, -1px -1px 4px #ddd inset;
    height: 32px;
    left: 4px;
    margin: auto;
    position: absolute;
    top: 0;
    transition: transform .5s ease;
    width: 32px;
}

.toggle-switch.active {
    transform: translateX(40px);
}

JavaScript

const switchOuters = Array.from(document.querySelectorAll(".switch-outer"));

switchOuters.forEach((outer) => {
    outer.addEventListener("click", () => {
        const dataAttribute = outer.dataset.setting;
        const isActive = outer.classList.toggle("active");
        const toggleSwitch = outer.querySelector(".toggle-switch");

        toggleSwitch.classList.toggle("active", isActive);
        document.body.classList.toggle(dataAttribute, isActive);
    });
});

解説

  1. 構造
    2. <div class="switch-outer" data-setting="darkmode">
    <div class="toggle-switch"></div>
</div>

    「スイッチの外側」 + 「スイッチの内側」で構成します。
    data-setting 属性はHTML上意味を持ちませんが、スイッチのID代わりになるのと、JavaScript を動かすために指定します。

  2. トグルスイッチの装飾
    3. .switch-outer {
    background-color: #ddd;
    border-radius: calc(infinity * 1px);
    height: 40px;
    margin: 8px 0;
    position: relative;
    transition: all .5s ease;
    width: 80px;
}

.switch-outer:hover {
    cursor: pointer;
}

.switch-outer.active {
    background-color: #0f0;
}

.toggle-switch {
    background-color: #fff;
    border-radius: 50%;
    bottom: 0;
    box-shadow: 1px 1px 8px #ccc, -1px -1px 4px #ddd inset;
    height: 32px;
    left: 4px;
    margin: auto;
    position: absolute;
    top: 0;
    transition: transform .5s ease;
    width: 32px;
}

    border-radius の仕様として、height の半分以上を指定しないと楕円形のような形になってしまいます。
    ただし絶対値を指定するとその都度変える手間が発生するので、常にheightの半分以上になるようcalc()で指定します。

    box-shadow でボタンに影をつけて立体感を出しています。
    外側: 右と下方向に1pxぶんズレた位置に、8pxぶんの強いぼかしが入ります。
    内側: 左と上方向に1pxぶんズレた位置に、4pxぶんのぼかしを inset で内側に入れます。

  3. トグルスイッチのアニメーション
    4. .toggle-switch.active {
    transform: translateX(40px);
}

    40pxぶんの横移動をさせます。
    40px = ボタン幅32px + (左右余白4px * 2)

  4. (例)ダークモードの反映
    5. body.darkmode {
    background-color: #333;
    color: #fff;
}

    body タグにそれぞれの data-setting 属性の値をクラスとして付与することで、
    ページ全体にCSSが反映されるようになっています。

  5. JavaScriptではdata-setting属性とスイッチ内側のDOMを取得し、クラスの付け外しを行う
    6. switchOuters.forEach((outer) => {
    outer.addEventListener("click", () => {
        const dataAttribute = outer.dataset.setting;
        const isActive = outer.classList.toggle("active");
        const toggleSwitch = outer.querySelector(".toggle-switch");

        toggleSwitch.classList.toggle("active", isActive);
        document.body.classList.toggle(dataAttribute, isActive);
    });
});

    HTML側で指定した data-setting 属性を取得し、変数 dataAttribute に代入します。
    代入された変数は body タグに付与するクラス名として使用します。

    スイッチの外側に .active クラスを付与して、背景色を灰→緑に変えます。

    要素の取得方法は様々ありますが、querySelector() は()内のセレクタに当てはまる最初の要素のみを取得する性質があるため、
    その性質を利用してスイッチ内側のDOMを取得することで、添字依存を解消しつつ確実に1つのスイッチのみを動かすことができます。
